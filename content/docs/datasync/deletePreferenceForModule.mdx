title: Delete Module Preference
description: >-
  This API deletes the configured preference setting of a module, which is
  utilized for syncing data between Zoho Desk modules and modules of other
  services. Note: Typically, a module can only sync with another single module.
  If you wish to sync a module that has already been synced with another service
  module, you must first delete the configured preference setting, then
  reconfigure and sync the module.
