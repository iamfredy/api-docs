---
title: Webhooks
description: Learn about the Help Center and its key support modules in Zoho Desk.
---
Zoho Desk supports webhooks that asynchronously deliver information on events that occur in the help desk. These events include actions, such as creating or updating a ticket, contact, or account, and so on. For information on how to include Zoho Desk webhooks in your service, refer to this [document](https://desk.zoho.com/support/WebhookDocument.do#Webhook).
